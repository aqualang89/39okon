## Решения

### 2026-04-19 Hero: PNG → WebP
**Решение:** конвертировали hero-bg.png (2.3МБ) в WebP (149КБ)
**Почему:** hero не грузился на мобилке и через ТГ-ссылки — слишком тяжёлый файл. WebP в 16 раз легче при том же качестве
**Доп:** добавлен preload в head, плавное появление фона через CSS-анимацию

### 2026-04-19 Убран WhatsApp
**Решение:** убрана ссылка на WhatsApp из секции контактов
**Почему:** заказчик попросил убрать полностью

### 2026-04-19 Overscroll + мобильный адаптив
**Решение:** html background → var(--dark), overscroll-behavior: none, отключены hover-эффекты на touch
**Почему:** при потягивании страницы вниз на мобилке виднелся белый фон body за хедером. Hover-анимации на карточках тормозили на телефонах

### 2026-04-19 Конфигуратор: реалистичный визуализатор
**Решение:** полностью переписан SVG-рендер окна
**Что добавлено:** откосы (трапеции с перспективой), 3D-подоконник с капельником, реалистичные ручки (розетка + стержень + набалдашник), блики на стекле, внутренние рамы створок, тени
**Почему:** старый визуализатор слишком простой — плоские прямоугольники без деталей, не убеждает клиента

### 2026-04-19 Конфигуратор: выбор цвета ручки, подоконника, откосов
**Решение:** добавлены 3 новых группы цветов — ручка (серебро/золото/бронза/чёрный/белый), подоконник (белый/мрамор/дуб/орех/антрацит/венге), откосы (белый/кремовый/серый/под цвет рамы)
**Почему:** конфигуратор должен давать полную картину — не только раму и стеклопакет

### 2026-04-19 Портфолио: убраны выдуманные подписи
**Решение:** удалены overlay-подписи типа "Квартира, ул. Сельма" из раздела "Наши работы"
**Почему:** названия были выдуманные, не соответствовали реальным объектам заказчика

### 2026-04-19 Cache bust через ?v= на CSS/JS
**Решение:** добавлен query параметр `?v=N` к style.css, main.js, configurator.js. Бампать при каждом пуше
**Почему:** без хешей в именах файлов браузер и ТГ-браузер кешируют старые версии. immutable убрали, must-revalidate стоит, но ?v= — гарантия

### 2026-04-19 Мобильный адаптив: overflow-x + отзывы
**Решение:** overflow-x: hidden на все секции, карточки отзывов уменьшены (шрифт, padding), max-width ограничен
**Почему:** страница гуляла влево-вправо, отзывы не влезали в экран. Дёрганье в ТГ-браузере — их баг, не наш
